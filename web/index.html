TV Underground Downloader



Introduction / Project presentation

The program permit to check Tv Serie on TvUndeground and automatically download via eMule.

Download
Last version is available here https://sourceforge.net/projects/tvudownloader/files/

Require
   1. Microsoft .Net 3.0

Stat up
   1. go in Activity tab and click “Options”
   2. set Url (ex : http://localhost:4000 or http://xxx.homeip.net:4000). Test the url using web browser
   3. set password ( see emule configuration)
   4. click “verify configuration”, if all parameter are correct you will receive a successful message
